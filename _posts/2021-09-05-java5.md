# 5. 경우에 따른 처리

## 5-1. if 문

```java
if(조건){
    문장1;
    문장2;
}
```

<br>

## 5-2. if~else 문

```java
if(조건){
    문장1;    
}
else{
    문장2;    
}
```

<br>

## 5-3. 여러 조건을 판단하기

```java
if(조건1){
    문장1;    
}
else if(조건2){
    문장2;    
}
else{
    문장3;
}
```

<br>

## 5-4. switch문

**식이 값1/값2/...일 경우**

```java
switch(식){
    case 값1:
        문장1;
        break;
    case 값2:
		문장2;
        break;
	default :
        문장D;
        break;
}
```

<br>
