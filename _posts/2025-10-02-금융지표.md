# 투자 성과 지표

## 1. Cumulative Return (CR)

포트폴리오의 전체 누적 수익률

$CR_t = \prod_{i=1}^{t}(1 + r_i) - 1$.

- $r_i$: $i$번째 일(day)의 포트폴리오 일별 수익률

- 의미: 모든 기간 동안 투자했을 때 **얼마나 벌었는가**
- e.g., CR = 0.10 → 총 10% 수익

<br>

## 2. Sharpe Ratio (SR)

“위험 대비 수익률”을 나타내는 **효율성 지표**

$SR = \frac{\bar{r}}{\sigma_r} \sqrt{252}$

- $\bar{r}$: 평균 일간 수익률
- $\sigma_r$: 일간 수익률의 표준편차 (변동성, risk)
- 252는 1년 거래일 수로 annualization 보정

<br>

즉, **Sharpe Ratio가 높을수록 “리스크 대비 효율적인 투자”**입니다.

<br>

## 3. Maximum Drawdown (MDD)

포트폴리오가 **가장 크게 떨어진 낙폭** (리스크 지표)

$MDD = \min_t \left(\frac{V_t}{\max_{s \le t} V_s} - 1\right)$

- $V_t$: 시점 t에서의 포트폴리오 가치
- 의미: **최고점 대비 최저점에서 얼마나 손실이 났는가**

<br>

예시:

- MDD = -0.25 → 최고점에서 25% 손실
- MDD가 작을수록 안정적인 포트폴리오

<br>

## 4. SR detail

퀀트 투자에서 가장 자주 쓰이는 **위험 대비 수익률 지표**

$SR = \frac{\bar{r} - r_f}{\sigma_r} \sqrt{252}$

- $\bar{r}$: 포트폴리오의 **평균 일간 수익률 (mean daily return)**
- $r_f$: **무위험 수익률 (risk-free rate)** → 보통 단기 국채 금리 (논문에서는 생략됨)
- $\sigma_r$: **일간 수익률들의 표준편차 (daily volatility)**
- $252$: 1년 거래일 수로, SR을 연 단위로 환산하기 위한 보정

<br>

표준편차(σᵣ) = **“수익률의 변동성(volatility)”**

Sharpe Ratio의 핵심:

- “수익이 평균에서 얼마나 흔들렸는가(=위험)”를 **표준편차**로 측정
- “수익률 시계열의 분산 정도”

<br>

| **일자** | **일간 수익률(%)** |
| -------- | ------------------ |
| Day 1    | +0.5               |
| Day 2    | -0.3               |
| Day 3    | +0.2               |
| Day 4    | +0.7               |
| Day 5    | -0.1               |

- (1) 평균 수익률: $\bar{r} = \frac{0.5 - 0.3 + 0.2 + 0.7 - 0.1}{5} = 0.2\%$

- (2) 표준편차 (변동성): 

  - $\sigma_r = \sqrt{\frac{(0.5-0.2)^2 + (-0.3-0.2)^2 + (0.2-0.2)^2 + (0.7-0.2)^2 + (-0.1-0.2)^2}{5}}$

    $= \sqrt{\frac{(0.3)^2 + (-0.5)^2 + 0 + (0.5)^2 + (-0.3)^2}{5}} = 0.41\%$

- 요약: 즉, 수익률은 평균 0.2%인데, 하루마다 ±0.4% 정도로 흔들린다는 뜻이에요.

- (3) $SR = \frac{0.2}{0.41} \sqrt{252} \approx 1.54$
  - Sharpe Ratio = **1.54** → 꽤 좋은 수치입니다.
  - 일반적으로 1 이상이면 효율적, 2 이상이면 매우 우수한 전략

<br>

해석

| **SR 값** | **의미**                                    |
| --------- | ------------------------------------------- |
| SR < 0    | 수익률이 마이너스거나, 변동성이 지나치게 큼 |
| SR ≈ 1    | 위험 대비 적절한 초과수익 (양호)            |
| SR ≥ 2    | 매우 안정적이고 수익 효율적인 포트폴리오    |
| SR ≥ 3    | 거의 “퀀트 신호급” (상업적 가치 높음)       |
