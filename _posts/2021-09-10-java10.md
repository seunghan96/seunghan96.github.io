# 10. 클래스 이용

## 10-1. 클래스 라이브러리

처음부터 직접 만들 필요 X

Java의 표준 개발환경인 JDK에는 **클래스 라이브러리**라는 많은 클래스들이 포함되어 있음

ex)

```
IOException
BufferedReader
InputStreamReader
System
String
Integer
...
```

<br>

해당 클래스에서, 다음과 같은 변수를 선언하여 사용하고 있음

```
br : Buffered Reader 클래스의 변수
str : String 클래스의 변수

br.readLine() : BufferedReader 클래스의 "인스턴스 메소드"
Integer.parseInt(str) : Integer 클래스의 "클래스 메소드"
```

<br>

## 10-2. 문자열 처리 클래스

대표적인 클래스 : **String 클래스**

주요 메소드

```
charAt, endsWith,.....
```

<br>

### charAt() & length() 메소드

```java
String str="Hello";
char ch1=str.charAt(0);
char ch2=str.charAt(1);
int len = str.length();
```

<br>

### 문자열 객체 생성 시, 주의점

객체 생성 시, "new" 연산자를 사용?

```java
// new 사용 O

// new 사용 X

```

<br>

### 대&소문자

```java

string str_U = str.toUpperCase();

```

<br>

### 문자 검색


## 10-5. 객체 배열

```java
class Sample
{
    public static void main(String[] args)
    {
        // 배열 준비
        Car[] cars;
        cars = new Car[3];
        
        // 배열에 요소 대입
        for(int i=0;i<cars.length; i++){
            cars[i] = new Car();
        }
    }
    
    // 생략
}
```
